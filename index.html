8D Corrective Action Report
D1 — Team
Champion: [Manager / 部門] Team Leader: [Your name] Members: [QA, Production, Engineering]
D2 — Problem Description
What: [不具合内容] Where: Found at [発見場所] When: First reported on [日付] How many: [不良数 / 影響ロット数]
D3 — Containment (24hr)
1. Isolated all suspect parts at [場所] on [日時]. 2. Sorted [数量] parts. Result: [OK数 / NG数]. 3. No nonconforming parts remain in transit. Effectiveness verified by: [確認者]
D4 — Root Cause
Occurrence cause: Why 1→2→3→4: [なぜなぜ展開] Root cause: [根本原因] Escape cause: Root cause: [流出根本原因]
※ 発生原因と流出原因は必ず分ける
D5–D6 — Actions
For occurrence: [恒久対策] Owner: [担当] By: [日付] For escape: [流出防止策] Owner: [担当] By: [日付] Documents updated: CP Rev.[No], FMEA Rev.[No]
D7 — Lessons Learned
Similar processes reviewed: [水平展開先] Actions: [内容]
D8 — Closure
Closed on [日付]. Approved by: [責任者名] All corrective actions verified as effective.
4M Change Notification
Header
CN No.: [CN-2024-XXX] | Supplier: [自社名] | Customer: [顧客名] Part No.: [品番] | Date: [日付] | Target: [希望実施日]
Change Description
Category: [Machine / Method / Material / Man] Current: [現状] Proposed: [変更後] Reason: [理由]
Impact Assessment
Product quality: [None / Yes — describe] Control Plan parameters: [None / CP will be revised] Special Characteristics: [None / Yes — describe] Validation plan: [試生産数・検査内容・判定基準]
Request
We kindly request your approval by [回答期限]. Contact: [氏名 / Email / Tel]
Audit Finding — Nonconformity (NC)
Header
Finding No.: [NC-01] | Clause: IATF 16949 [条項番号] | Class: [Minor/Major]
Statement
Objective evidence shows that [具体的な事実]. This does not conform to clause [条項], which requires: "[条項の要求内容]"
※ 事実のみ。意見・推測を入れない
Evidence
- [記録名・文書番号] - Interview with [役職・氏名] on [日付]
Required Action
Submit corrective action plan by [日付] addressing: 1. Root cause 2. Corrective action 3. Evidence of implementation
Audit Finding — OFI
OFI Statement
During the audit, it was observed that [観察事実]. While this does not constitute a nonconformity, the audit team suggests that [改善方向] could further strengthen the QMS.
※ OFIはsuggestion。"shall"は使わない。"may / could / consider"を使う
Potential Benefit
- [期待効果1] - [期待効果2]
